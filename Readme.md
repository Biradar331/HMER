### Installation Workflow
1. Create the virtual env for the project.
2. open the project in env with the following requirements
    python - 3.6.8
    pytorch == 1.0
3. Install all the necessary modules using pip ...
4. Download the model 
5. unzip the off_image_train and off_image_test
6. test the model by running the densenet_testway.py 
7. Run the finalTK.py for the interface of the final project(Prediction system)
8. save the results in training_data folder

### To Work in Virtual ENV
1. Go to Virtual ENvironment using cmd as venv\scripts\activate
